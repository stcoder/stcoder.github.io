Ssh

Подключение

Генерация приватного ключа:

$ ssh-keygen -t rsa -C "<комментарий>" -f <file_name> -P "<passfraze>"

Копируем публичный ключ на удаленный сервер:

$ ssh-copy-id -i <identity_file_name> <hostname>

Теперь для подключения к серверу выполняем команду:

$ ssh <hostname>
