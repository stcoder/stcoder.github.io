Git

Тут буду вести всякие заметки по git.

Переименование ветки

git branch -m old_branch new_branch - переименовать локальную ветку

git push origin :old_branch - удалить старую ветку

git push -u|--set-upstream origin new_branch - запушить новую ветку на удаленный сервер

Отмена мержа

git checkout master - переходим локально в ветку master

git reset HEAD^ - отменяем последние изменения

git push -f origin master:master - обновляем ветку на удаленном сервере

Удалить все замерженный локальные ветки

git branch --merged | egrep -v "^\*|master|develop" | xargs git branch -d

  • git branch --merged - вернет все замерженные ветки
  • egrep -v "^\*|master|develop" - исключит из списка текущую ветку + master и develop
  • xargs git branch -d - выполнит действие на полученным списком веток, в данном примере - просто удалит их.

Удалить непроиндексированный файлы/папки

git clean -n - покажет список файлов которые находятся вне индекса.

git clean -nd - удалит файлы

git clean -fd - удалит директории

Очищает репозиторий (приводит его к состоянию git clone)

git clean -ndx && git clean -fdx

Тэги

git tag -a <tagname> -m "<message-tag>" - создать тэг локально

git push origin <tagname> - запушить тэг на сервер

git tag -d <tagname> - удалит тег локально

git push --delete origin <tagname> - удалит тэг с сервера
